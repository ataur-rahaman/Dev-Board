Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the dom. Such as:
getElementById
getElementsByClassName
getElementsByTagName
querySelector etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: We can get all the tags and texts from an element.
innerText: We can get only the visible text from an element.
textContent: We can get all text including hidden text(id name or class name) from an element.

Question-3: What is event delegation in the DOM?

Event delegation is a way to handle events more efficiently. Instead of adding event listeners to many child elements, we add one event listener to parent element. When a child element is clicked, the event moves up to the parent. This makes our code faster and works even if new child elements are added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process when an event starts from the clicked element and moves up through its parent elements one by one, until it reaches the top
